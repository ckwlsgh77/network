# http (응용계층)


- client가 보내는 요청에 server가 응답 
- 응답처리후 연결 종료
- 단방향적 통신
- 비연결성 프로토콜 -> 요청/응답 방식
- 도메인 + URL, 도메인 + URI 를 통해서 요청, HTML을 통한 응답
- Cookie 와 Session 이용
